# SvelteKitTemplate

A starter template using SvelteKit 2, Svelte 5, TypeScript, Tailwind CSS 4, and Vite 8.

Pre-configured with ESLint, Prettier, Husky, and lint-staged.

## Prerequisites

- Node.js 18.17+ (Node 20+ recommended)
- npm

## Setup

```npm
npm install
```

## start the development server
```npm
npm run dev    
```
# create a production build 
```npm
npm run build
```    

Use `npm run preview` to locally preview the production build.

## JetBrains IDE

This template is intended to be used with JetBrains IDEs (WebStorm, IntelliJ IDEA Ultimate). Ready-to-use run configurations are included in `.idea/runConfigurations`, so you can start the dev server or run a build directly from the IDE's Run widget without any extra setup.
